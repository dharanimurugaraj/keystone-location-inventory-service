# Keystone Commerce Backend Assignment

Location-Based Inventory Reservation Service

## Overview

A backend service built using **NestJS**, **TypeScript**, **PostgreSQL**, and **Prisma** to model location-based inventory reservation during checkout while ensuring correctness under concurrency, idempotency, and payment state transitions.

This project is being developed incrementally following the documents in the `docs/` directory.

## Tech Stack

- NestJS
- TypeScript
- PostgreSQL
- Prisma ORM
- Docker
- Jest
- Swagger (OpenAPI)
